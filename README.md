# East Asian Soups and Stews
A multi-page static website created for a web development course at Sierra College. The site focused on East Asian soups and stews while demonstrating core HTML and CSS skills learned through the class.

## Live Demo
https://thanatos808.github.io/Soups-and-Stews-website/

## Project Overview
This website was designed as a final web development project to showcase foundational front-end development skills. The course focused primarily on HTML and CSS, so the site was built as a static website without Javascript or backend functionality.

## Features
- Multi-page website structure
- Semanting HTML layout
- Consistent navigation across pages
- Responsive page styling
- Images
- Styled header, footer, main content, and aside functions
- Recipe and informational content sections

## Notes
This project was built for educational purposes to demonstrate HTML and CSS skills learned. It was not intended to showscase dynamic JavaScript functionality or advanced visual/UI design, though the project scope could be expanded later to practice those areas.
